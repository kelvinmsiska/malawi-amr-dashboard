Upload new facility data

The dashboard reads live from Google Sheets. To update with a new reporting period or a new facility, follow these steps:

1
Export your .numbers or Excel file as CSV and place it in the shared Google Drive folder.
2
Open AMR_Dashboard_Pipeline.ipynb in Google Colab. Set FACILITY_CODE to your site code (e.g. "KCH"). Click Run all.
3
The pipeline cleans the data, tags geographic fields from the facility master list, and pushes to Google Sheets automatically.
4
This dashboard reads from Google Sheets and auto-updates within 15 minutes. No further action needed.

The dashboard dynamically detects all facilities, regions, districts, and organisms present in the data — no code changes needed when new sites onboard.
